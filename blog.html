Question 4

It is better to use relative size when styling elements for a responsive website as opposed to absolute size because it lets browsers on different devices, customize the web page and its elements to their specific viewport. Styling using relative size also allows users to have a smoother experience while zooming in because the web page readjusts itself to make the content visible. Zooming out makes it tough to see the content, especially characters because their size gets smaller the more we zoom in.

16px is the default width browsers give to elements and not the actual size of any screen. When a browser receives an instruction to set an element to 200px it changes the default value for that specific element and displays it on the screen.
